# Gitty

Gitty is a simple web based Git Client.

## Features

- Cloning
- Opening a local repository
- Commiting(`git add` and `git commit`) with a message
- Pushing
- Logging in a `.log` file for easy debugging

## Development

### Create a virtual environment

```
python3 -m venv .venv
source .venv/bin/activate
```

### Install Dependencies(UV)

```
uv sync
```

### Run project

```
flask run --debug
```
